# 构建过程

记录该前端脚手架的构建过程。

## 安装vite3.0.0

https://cn.vitejs.dev/guide/#scaffolding-your-first-vite-project
```
npm create vite@latest my-app -- --template react-ts
```

## 声明npm包使用的镜像源
添加文件`.npmrc`

```
registry=https://registry.npm.taobao.org 
```

## 第一次安装依赖

```
npm i
```


## 安装gts（eslint & prettier）

```
npx gts init
# tsconfig.json 不能被覆盖
```
